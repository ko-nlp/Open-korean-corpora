# Changelog

## Third edition — Jun. 2026

This release reflects the updated survey manuscript (TALLIP submission). Highlights:

### Structure
- Documentation status (*Docu.*) redefined: `docu / paper / int'l` → **`int'l / dom. / none`** (internationally available / domestic only / no official description).
- Redistribution (*Redist.*) values simplified to **`rd / none / unk`**.
- Category 4 split into two: **Intention and sentiment** and **Offensive language detection, fairness and bias** — total categories 9 → 10.

### Newly added datasets (mostly 2023–2025)
- **Benchmark studies**: Open-Ko-LLM, HAE-RAE Bench, KMMLU, KULTURE Bench, KMMLU-Redux/Pro, KoBALT.
- **Intention and sentiment**: KMRE, ToM-Diary, KEmoFact, KPC-cF, KoCoSa, KOTE, CARBD-Ko, KPoEM.
- **Offensive language, fairness and bias** (new category): KoMultiText, K-HATERS, KoSBi, SQuARe, KoBBQ, KCDD, LifeTox (alongside existing BEEP!, APEACH, Unsmile, HateScore, KOLD, K-MHaS, DKTC, KODOLI).
- **Entailment / similarity / paraphrase**: Korean PPDB, KoSEnd.
- **QA and dialogue**: KorWikiTQ, CLIcK, KoDialogBench, KorNAT, K-MMBench, K-Viscuit, KoSimpleQA, KoPIQA.
- **Summarization, translation, transliteration**: OPUS-MT ko-en, Naver News Summary, KoreaScience Summary, SSL (sign language), KPC, KNOTICED.
- **Korean in multilingual corpora**: XL-Sum, MASSIVE.
- **Speech corpora**: OLKAVS, KMSAV.
- **Other topics**: KorMedMCQA, KBMC, ESG-Kor, KBL, FunctionChat-Bench, KCL, KorMedLawQA.

### Maintenance
- Refreshed broken/moved links (e.g. OpenKorPOS, KSS, MultiCoNER), corrected volume figures (e.g. NSMC, Sci-news-sum-kr), fixed publication years, and merged closely related entries (KorQuAD 1.0/2.0, KorNLI/KorSTS, AIR NER/SRL).
